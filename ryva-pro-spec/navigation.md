# Navigation

## Desktop shell

### Primary sidebar

1. Home
2. Products
3. Brands
4. Businesses
5. Placements
6. Outreach
7. Accounts
8. Orders
9. Reorders
10. Commissions
11. Analytics

### Utility area

- Global Search
- Create
- Tasks
- Calendar
- Documents
- Notifications
- Help
- Profile and Settings

The sidebar supports compact and expanded modes. Section order is fixed in the first version to preserve training consistency.

## Global Create

Creates:

- Product research record;
- Brand;
- Business;
- Contact;
- Representation Opportunity;
- Placement Opportunity when prerequisites exist;
- Task;
- Note;
- Order;
- Document upload.

Unavailable actions explain the prerequisite rather than disappearing silently.

## Record header

Every detail page shows:

- record type and name;
- state/status;
- owner;
- critical warning or conflict;
- last meaningful activity;
- next action;
- primary action;
- related-record switcher;
- history/evidence drawer.

## Breadcrumbs

Use hierarchy only where real: Brands → Brand → Representation Agreement. Cross-linked records use a Back link plus relation context rather than false nesting.

## Mobile navigation

Bottom navigation:

- Home
- Tasks
- Placements
- Search
- More

More exposes Products, Brands, Businesses, Outreach, Accounts, Orders, Reorders, Commissions, Calendar, Documents, Analytics summaries, and Settings.

## Command palette

Desktop keyboard command palette supports navigation, record lookup, new internal records, task completion, note creation, and safe internal actions. It cannot send outreach, approve agreements, change credential state, or mark commission paid without entering the relevant reviewed workflow.

## Context preservation

Filters, columns, sort, scroll, and selected saved view persist per user and workspace. Opening a record in a drawer or new tab does not lose list context.
