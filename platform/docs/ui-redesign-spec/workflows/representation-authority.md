# Representation Authority

- **Purpose:** Show how Brand diligence becomes written, scoped, human-approved authority.
- **Used on:** Representation register/detail, Brand detail, Placement/Outreach authority context.
- **Stages:** Discovered → Researching → Contact ready → Contacted → Conversation → Reviewing terms → Authorized → Active; Paused/Ended/Rejected are explicit terminal/branch states.
- **Completed state:** Entry and exit criteria are satisfied and recorded by stage event.
- **Current state:** Current Representation Opportunity stage.
- **Blocked state:** Missing identity/contact/decision/task, incomplete terms, unsigned/unapproved Agreement, ambiguity/conflict, expiry/suspension/termination.
- **Required next action:** Existing owned task or exact missing criterion.
- **Authority gate:** “Authorized” requires an active reviewed Agreement with products/scope/authority; upload/extraction is never a completed gate.
- **Visual form:** Linear stepper for current path; terminal/paused states shown as labelled branch, not floating nodes.
- **Accessibility alternative:** Stage table with entry criteria, completed criteria, missing criteria, state, next action.
- **Mobile alternative:** Current stage card + vertical prior/next steps; no horizontal canvas.
- **Acceptance criteria:** Human approval and exact artifact remain mandatory; pause/end block Outreach; every transition uses server validation and audit.
