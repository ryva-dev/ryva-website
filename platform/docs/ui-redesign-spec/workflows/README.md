Not selected:

- ordinary record qualification lists;
- generic task completion;
- analytics;
- commission calculation internals;
- evidence classification;
- settings and admin jobs.

Those are clearer as tables, timelines, formulas, or status summaries.

## Shared state language

- **Completed:** requirement is satisfied by current valid evidence; check icon + “Completed.”
- **Current:** active stage/review; accent marker + “Current.”
- **Available:** permitted next stage; neutral outline + action label.
- **Blocked:** cannot advance; danger/warning icon + reason and required action.
- **Expired/stale:** previously complete requirement no longer current; warning text and date.
- **Not applicable:** explicit text, not a hidden node.

Each visualization has an ordered list/table carrying identical information and actions.
